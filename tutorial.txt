Initial Tutorial

To run a simple game:

1) Your HTML must contain:
- a div with id `StartRoomLoad` (room image/video)
- a div with id `StartRoomText` (narrative text)
- an input that calls `iterateCmds(value)`.

2) Include files in this order:
- includes/globals.js
- includes/globalmessages.js
- includes/functions.js
- includes/synonyms.js
- includes/parser.js
- engine/core.js
- engine/usercommands.js
- gamedata/game.js

3) Minimal HTML example:

```html
<div id="StartRoomLoad"></div>
<div id="StartRoomText"></div>
<input type="text" onkeydown="if(event.key==='Enter'){iterateCmds(this.value);this.value='';}">
```

4) Example game data is now provided in `gamedata/game.js`.
- Start room: `foyer`
- Try commands: `look door`, `east`, `north`, `look lamp`, `inventory`


Inventory scripting:
- Use `addItemToInventory('item_id', 'Display Name')` in room handlers/scripts to add an item to player inventory.
- Use `inventory` / `inv` / `i` / `items` / `bag` to view carried items.
