Initial Tutorial

To run a simple game:

1) Your HTML must contain:
- a div with id `StartRoomLoad` (room image/video)
- a div with id `StartRoomText` (narrative text)
- an input that calls `iterateCmds(value)`.

2) Include files in this order:
- includes/globals.js
- includes/globalmessages.js
- includes/functions.js
- includes/synonyms.js
- includes/parser.js
- engine/core.js
- engine/usercommands.js
- gamedata/game.js

3) Main runtime data structures:
- `GameLocations`: room/location data
- `GameLocationTravel`: exits/blocked state per location
- `GameObjects`: room item data (items are separate from locations)
- `GameObjectCommands`: item-specific command handlers

4) Item structure example:

```js
lamp: [
  ['Item Name', 'lamp'],
  ['Id', 'lamp'],
  ['Location', 'foyer'],
  ['Unique Name', 'foyer lamp'],
  ['Object Type', 'item'],
  ['Description', 'An oil lamp.'],
  ['Alt names', 'hot lamp'],
  ['Alt names 2', 'oil lamp'],
  ['Can be Taken', 'Yes']
]
```

5) Change an item field at runtime:
- `changeObjectValue('lamp', ITEMDESC, 'A lit lamp');`

6) Inventory:
- `inventory` / `inv` / `i` / `items` / `bag`
- Add item via script: `addItemToInventory('item_id', 'Display Name')`
- If an item has `Can be Taken = Yes`, `take <item>` works automatically.
- `look <direction>` can show directional look text from `GameLocationTravel` (eg: `look east`).
